# LIMITLESS – Anime Inspired Fitness Landing Page

## Overview

LIMITLESS is a responsive landing page created using HTML and CSS with an anime-inspired fitness theme. The idea behind the project was to combine anime-style motivation with a modern fitness landing page design.

## Development Process

I first created the basic HTML5 structure in `index.html` and linked it with `style.css`. The webpage was divided into three sections: header, hero section, and footer.

The header contains the logo and navigation links. The hero section includes a motivational heading, short description, and a call-to-action button. The footer contains social media links to complete the layout.

## Styling and Design

The webpage was styled entirely using CSS. I used a dark gradient background with neon-inspired colors to match the anime fitness theme. Flexbox was used for alignment and spacing throughout the page.

Hover effects were added to navigation links, buttons, and social links to improve user interaction. CSS transitions, padding, margins, and border radius were also used to create a cleaner design.

## Responsive Design

Media queries were implemented to make the webpage responsive on smaller screens. The navigation bar changes into a vertical layout on mobile devices, and font sizes adjust automatically for better readability.

The responsiveness was tested by resizing the browser window and using device view in browser developer tools.

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries

## Learning Outcome

This project helped me understand webpage structure, responsive design, Flexbox layouts, and styling techniques using HTML and CSS.
